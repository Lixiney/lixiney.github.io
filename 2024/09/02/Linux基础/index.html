常用基础命令

命令 作用
ls 显示目标列表
cd [path] 切换工作目录
pwd 显示工作目录的绝对路径
echo [string] 打印一行文本
cat [file] 连接文件并打印到标准输出设备上
less [file] 允许用户向前或向后浏览文字档案的内容
mv [file1] [file2] 用来移动或者重命名一个文件或目录
cp [file1] [file2] 用来复制一个文件或目录
rm [file] 可以删除一个文件或目录
top 实时查看系统运行状态
ps 当前系统的进程状态
kill 杀死一个进程
ifconfig 查看或者设置网络设备
ping 查看与网络主机的连通性
netstat 显示网络连接、路由表和网络接口信息
su 切换用户身份
touch [file] 创建空文件
mkdir 创建空目录
chmod 变更文件或目录的权限
nano / vim 字符终端的文本编辑器
clear 清除终端
exit 退出终端

Bash 快捷键

快捷键 作用
Up (Down) 上（下）一条指令
CTRL + C 终止当前进程
CTRL + Z 挂起当前进程 使用 “fg” 唤醒
CTRL + D 删除光标处的字符
CTRL + L 清屏
CTRL + A 移动到命令行首
CTRL + U 清除当前行
CTRL + SHIFT + C 复制
CTRL + SHIFT + V 粘贴

目录结构

  • /bin 、/sbin : 存放linux的一些核心二进制文件，其包含的命令可以在 shell 下运行
  • /boot ：操作系统启动时要用到的程序
  • /dev ：包含了所有 Linux 系统中使用的外部设备的端口
  • /etc : 存放各种配置文件
  • /home ：普通用户的主目录
  • /lib 、 /lib64 ：存放一些动态链接共享库
  • /mnt ： 临时挂载其他文件系统
  • /root ：系统管理员的主目录
  • /usr ： 应用程序和文件几乎都在此文件夹
  • /var ： 存放了很多服务的日志信息
