    项目名称 Usage 地址

    x64dbg / x32dbg 在 windows 上使用的开源 x64 / x32 调试器 https://x64dbg.com/

    dnSpy 强大的 .NET 调试、修改和反编译的工具 (已停止维护) https://github.com/dnSpy/dnSpy

    项目名称 Usage 地址

    项目名称 Usage 地址
